# 🎯 Roleta de Campeões - LoL

Uma roleta interativa de campeões de **League of Legends**, feita com HTML, CSS e JavaScript. Ideal para escolher campeões aleatórios de forma divertida 🎮

---


## 🚀 Funcionalidades

- 🎲 Sorteio aleatório de campeões
- 🌀 Animação de rotação ao sortear
- 🖼️ Exibição da imagem e nome do campeão
- 🎵 Música de fundo via YouTube (player invisível)
- 🔊 Controle de volume
- 🌌 Background customizado com overlay escuro
- 📱 Interface simples e responsiva
